# fundWap-app

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for prod with minification
npm run build

# build for prod and view the bundle analyzer report
npm run build --report
```
